# What Professors Typically Ask

Use this as an assessment-alignment guide.

## Module 3.1

- Derive TF from a physical ODE.
- Distinguish command response vs disturbance response.
- Build or interpret Simulink block arrangement.

## Module 3.2

- Compute closed-loop transfer function.
- Use final-value theorem for SSE.
- Interpret system type and steady-state tracking limits.

## Module 3.3

- Read crossover and phase margin from Bode.
- Explain robustness vs speed tradeoff.

## Module 3.4

- Choose lead/lag/PID action for a stated requirement.
- Justify tuning direction using Bode effects.

## Module 3.5

- Build Routh table and infer stable parameter range.
- Apply Nyquist $N=Z-P$ with correct encirclement counting.

## Module 3.6

- Convert $C(z)$ to difference-equation implementation.
- Explain practical effects of computational/sampling delay.

## Module 3.7

- Check discrete stability using unit-circle/Jury logic.
- Explain sampled-data block roles (sampler, hold, plant).

## Module 3.8

- Select sample time from performance/aliasing considerations.
- Explain ZOH and converter impacts.

## Module 3.9

- Form state-space model from ODE.
- Use state-transition matrix interpretation.

## Module 3.10

- Derive or apply exact discretization ($A_d$, $B_d$).
- Compare exact vs approximation methods.

## Module 3.11

- Compute controllability/observability matrices and rank.
- Apply Ackermann formula for pole assignment (SISO).

## Module 3.12

- Design controller poles and observer poles.
- Explain separation principle and practical pole-speed choices.

## Module 3.13

- Compare LQR and MPC for constrained systems.
- Explain role of weighting matrices and horizon choices.

## Lab (Inverted Pendulum)

- Defend linearization assumptions.
- Justify controller and observer design choices.
- Explain failure modes and improvements from test logs.
