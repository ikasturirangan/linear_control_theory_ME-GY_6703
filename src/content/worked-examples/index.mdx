# Worked Examples

Each page shows complete steps, not just final answers.
Use them to learn how to structure your own solution flow.

## Suggested routine

1. Attempt the problem yourself for 5-10 minutes.
2. Compare with the full solution.
3. Identify exactly where your process diverged.
4. Re-solve without looking.
