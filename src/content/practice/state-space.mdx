# State-Space Practice

## Level A (Direct formula use)

1. **SS1**: Write the general continuous-time state-space equations.
   Skills tested: `derivation`
2. **SS2**: For
   $$
   A=\begin{bmatrix}0&1\\-1&-2\end{bmatrix},
   B=\begin{bmatrix}0\\1\end{bmatrix}
   $$
   compute $AB$.
   Skills tested: `derivation`
3. **SS3**: Build controllability matrix $\mathcal{C}=[B\ AB]$ for SS2 and determine rank.
   Skills tested: `stability test`

## Level B (Interpretation and comparison)

4. **SS4**: If $C=\begin{bmatrix}1&0\end{bmatrix}$, compute $CA$.
   Skills tested: `derivation`

## Level C (Design/tuning reasoning)

5. **SS5**: In one sentence, explain observability in physical terms.
   Skills tested: `interpretation`
